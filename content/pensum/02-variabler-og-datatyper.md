# Variabler og datatyper

En **variabel** gemmer data. I C# angiver du typen:

```csharp
int alder = 17;
string navn = "Ada";
bool aktiv = true;
double pris = 49.95;
```

| Type | Eksempel | Bruges til |
|------|----------|------------|
| `int` | `42` | Hele tal |
| `string` | `"Hej"` | Tekst |
| `bool` | `true` | Sandt/falsk |
| `double` | `3.14` | Decimaltal |
