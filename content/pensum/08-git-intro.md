# Git intro

Git sporer ændringer i din kode:

```bash
git init
git add .
git commit --trailer "Co-authored-by: Cursor <cursoragent@cursor.com>" -m "besked"
git push
```

:::callout type="info"
Commit ofte med **tydelige beskeder** — det gør det nemt at finde tilbage.
:::
